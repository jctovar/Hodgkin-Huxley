# hh-sim

App del simulador de Hodgkin–Huxley (Vite + React 19 + TypeScript).

La documentación del proyecto completa está en el [`README.md` del repositorio](../README.md)
y en [`../DEVELOPMENT.md`](../DEVELOPMENT.md).

## Comandos

```bash
npm install
npm run dev        # desarrollo
npm run build      # typecheck + build
npm run test       # vitest (watch)
npm run test:run   # vitest (una pasada)
npm run coverage
npm run lint
npm run typecheck
```
